Sign in with EVE Online

Every tool in this suite shares one sign-in. Add as many characters as you like — from the same corp or different ones — and each tool will use whichever applies.

To add a character from a different EVE account, log out at login.eveonline.com first (or use a private/incognito window) — EVE's SSO doesn't offer a way for this page to force that switch for you.
